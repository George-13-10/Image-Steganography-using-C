# 🖼️ Image Steganography in C

This project implements **Image Steganography**, a technique used to hide secret data inside an image without visibly altering it.

---

## 📌 📖 Description

Image Steganography is used to securely embed a secret message inside an image file.  
This project uses **LSB (Least Significant Bit) technique** to hide and extract data from `.bmp` images.

---

## 🚀 Features

- 🔐 Encode secret message into an image
- 🔍 Decode hidden message from image
- 📂 Supports BMP image format
- 🧠 Uses LSB algorithm
- ⚡ Efficient and lightweight implementation in C

---

## 🛠️ Technologies Used

- C Programming
- File Handling
- Bit Manipulation
- Data Structures (basic)
