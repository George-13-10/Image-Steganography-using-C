# 🔒 Image Steganography using Embedded C

A command-line based Image Steganography application developed in **Embedded C** that securely hides and retrieves secret text messages inside **24-bit BMP images** using the **Least Significant Bit (LSB)** encoding technique.

The project demonstrates low-level file handling, bitwise manipulation, pointer operations, and memory-efficient programming concepts commonly used in Embedded Systems and Firmware Development.

---

## 📌 Project Overview

Image Steganography is the process of hiding confidential information inside an image without noticeably altering its appearance.

This application performs:

- Encoding (Hide Secret Message)
- Decoding (Retrieve Secret Message)

using the Least Significant Bit (LSB) algorithm.

Unlike encryption, steganography conceals the existence of the message itself.

---

## 🚀 Features

- Encode text into BMP images
- Decode hidden text from BMP images
- Supports 24-bit BMP image format
- Lossless message retrieval
- Command-line interface
- Memory-efficient implementation
- File validation and error handling
- Modular Embedded C implementation
